# vue-mailup

MailUp plugin for VueJS

## Usage

`$ npm install vue-mailup`

```js
import Vue from 'vue';
import VueMailUp from 'vue-mailup';

Vue.use(VueMailUp, options)
```

## Options

Options are **mandatory**

```js
let options = {
  list: 1, // list ID
  key: 'k2z5.s01.it' // your console URL
};
